wangEditor demo byRequirejs

当你看到这个页面的时候，wangEditor已经开始支持AMD，可以是用requirejs来加载wangEditor。

首先，在页面中引用require.js

<!--默认启动页面为 main.js-->
<script data-main="main" src="require.js"></script>

main.js代码如下：

require.config({
    paths: {
        jquery: '../../dist/js/jquery-1.10.2.min',
        wangEditor: '../../dist/js/wangEditor-1.3.0.min'
    }
});

require(['wangEditor'], function(){
	$(function(){
		var editor = $('#textarea1').wangEditor();
	});
});

这样就OK啦，很简单吧！
