wangEditor demo


在demo-basic.html应用的基础上，可通过 html() 方法修改html内容。如下例子：

var editor = $('#textarea1').wangEditor();

$('#btnHtml').click(function(e){
	var dt = new Date();
	editor.html('<p>欢迎使用wangEditor，当前的时间是：' + dt.toString() + '</p>');

	//如果想要把editor清空，只需要传入空字符串即可，如下代码：
	//editor.html('');
});
